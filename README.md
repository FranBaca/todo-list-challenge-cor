# App Todo List 

Aplicación Todo List Challenge creada por Francisco Baca para COR.

- Se pueden crear y eliminar tareas.
- Se pueden modificar la tarea.
- Se pueden filtrar las tareas por estado.

## Instalación y ejecución

```sh
npm install

npm start
```

## Desarrollada con:

- React
- Tailwind CSS
